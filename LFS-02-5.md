## LFS: To Shrink and Clean the VirtualBox VDI File

### Fill the Guest disk with zero (root)

```
#!/bin/bash

FILE="ZZZZ"

cd /tmp
df .
dd if=/dev/zero of=$FILE bs=1M
df .
rm $FILE

cd /mnt/lfs/lost+found
df .
dd if=/dev/zero of=$FILE bs=1M
df .
rm $FILE

echo ""
echo "vboxmanage modifymedium --compact"
echo ""

exit
```
